# 📊 Análise de Vendas com Power BI

## 📌 Sobre o projeto
Dashboard desenvolvido no Power BI com foco em análise de dados de vendas, permitindo a visualização de indicadores estratégicos e apoio à tomada de decisão baseada em dados.

## 🎯 Objetivo
Analisar o desempenho de vendas e identificar padrões, tendências e oportunidades de melhoria através da visualização de dados.

## 🛠️ Tecnologias e ferramentas
- Power BI  
- Excel  
- Modelagem de Dados  
- ETL (tratamento e transformação de dados)  
- DAX (medidas e cálculos)

## 📊 Principais análises
- Faturamento total e por período  
- Produtos mais vendidos  
- Desempenho por região  
- Evolução mensal de vendas  
- Indicadores de performance (KPIs)

## 🧠 Insights gerados
- Identificação de regiões com maior volume de vendas  
- Produtos com maior contribuição no faturamento  
- Tendências de crescimento ao longo do tempo  
- Possíveis oportunidades de otimização comercial
